## Dart Canvas Player

[Demo](https://eatsjobs.github.io/dart-canvas-player/)

Install dependencies
```shell script
pub get
```

Run the webserver
```shell script
webdev serve
```

Generate the build
webdev build

Run tests with debugger
```shell script
pub run test -p chrome ./web/test/CanvasPlayer_test.dart --pause-after-load
```
